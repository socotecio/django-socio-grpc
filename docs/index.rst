Django Socio gRPC, Django Rest Framework for gRPC
=================================================

Welcome to the Django Socio GRPC (DSG) documentation! DSG is a Python
library that allows you to create gRPC services in a Django application.
It is similar to Django Rest Framework (DRF) and provides features
for setting up services and data serialization.

Getting Started
---------------

To get started with DSG, please refer
to the :doc:`installation guide <getting-started>`.
